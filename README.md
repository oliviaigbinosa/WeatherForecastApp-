PROJECT TITLE:  Weather Forecast App

Overview: This project is a simple Weather Forecast App to display the current weather conditions of locations worldwide.  It makes use of two files, " weather.py " to fetch weather data and " main.py " to handle input and output.

Repository URL: https://github.com/oliviaigbinosa/WeatherForecastApp-


Below is the full Software Development Life Cycle (SDLC) of the project:


1) Planning:
    - Scope or Purpose: Display current weather and forecast
    - Target users: Anyone needing weather updates

2) Analysis:
     It requires an API integration, a simple user interface (UI) and location input.

3) Design:
     It uses modules such as WeatherService to fetch data, user interface (UI) to display data and Data such as location, temperature, conditions, forecast etc

4) Implementation: this is given in the code (weather.py, main.py)

5) Testing: It tests API calls and UI updates

6) Deployment: It is pushed to GitHub

7) Maintenance:  Update API keys, add features
